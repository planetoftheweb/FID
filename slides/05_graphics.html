Bitmaps

  • AKA Raster Graphics
  • Grid of pixels
  • 3 dimensions: width, height, depth

Bitmap Depth

  • Depth = how many colors
  • More colors = richer depth
  • Bigger depth = bigger size
  • Great for Photographs

Vectors

  • Resolution independent
  • Mathematical formulas
  • Converted to Bitmaps

Web Formats

  • GIF
  • JPG
  • PNG
  • SVG

GIF

  • Oldest format
  • Up to 256 colors
  • 1-bit transparency
  • Supports animation

JPEG

  • 256 colors (8-bits) per channel
  • Perfect for photos
  • Highly compressed

PNG

  • Best quality (64-bits) per channel
  • Best Transparency
  • Not Lossy
  • Not as small as JPEG
  • Better at simple graphics

SVG

  • Great new format
  • Vector Format
  • Animation
  • CSS Styling

Reducing Image Size

  • Cameras take HUGE pictures
  • Crop or resize
  • Reduce pixel density

Reducing Size

  • PNG - small colors QTY
  • JPG for photos
  • Resize/crop
  • Thumbnails

Use Photoshop to re-compress your sample images. Create Thumbnails of images at least 300px x 200px
